What are Custom Schemas?

XMP metadata supports user-defined namespaces. Custom schemas let you define your own metadata fields (e.g. family:event, archive:source) that get embedded into exported XMP sidecar files alongside standard EXIF and Dublin Core fields.

  • Each schema has a namespace prefix (e.g. family) and a namespace URI (e.g. https://example.com/ns/family/1.0/)
  • Fields you define appear as editable inputs in the XMP Sidecar Viewer on each asset
  • Exported XMP files reference your namespace so any XMP-aware tool can read them
  • Supported field types: text, number, date, boolean, url
